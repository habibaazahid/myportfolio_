# 🌐 My Personal Portfolio Website

This is my **personal practice project** — a portfolio website I built while learning web development from scratch. I'm a beginner CS student working on improving my HTML, CSS, and JavaScript skills, and this project is part of that learning journey.


## 👩‍💻 About This Project

I built this website to:
- Practice writing real HTML, CSS, and JavaScript
- Learn how to structure a complete webpage
- Experiment with animations and interactive effects
- Get comfortable with VS Code, GitHub, and deployment
- Have a place to showcase my learning progress over time

This is not a perfect project — it's a **work in progress**, just like me! 


## 🛠️ What I Used

| Technology | What I Learned |
|---|---|
| HTML5 | Page structure, semantic tags, linking sections |
| CSS3 | Styling, flexbox, grid, animations, variables |
| JavaScript | DOM manipulation, events, scroll effects, timers |
| Google Fonts | How to import and use custom fonts |
| Vercel | How to deploy a website for free |
| GitHub | Version control and hosting code |



## ✨ Things I Tried Building

- A custom animated cursor that follows the mouse
- Scroll reveal animations (elements fade in as you scroll)
- Animated skill bars that fill up on scroll
- SVG circular progress rings
- A typing animation that cycles through text
- Floating badges in the hero section
- Responsive layout that works on mobile too

Some of these were challenging for me as a beginner — but that's the point! 🚀
