Personal Portfolio Website
A fully responsive, animated personal portfolio website built with HTML, CSS, and vanilla JavaScript as part of my personal learning journey into web development.
🔗 Live Site: myportfolio-sk7e.vercel.app

👩‍💻 About This Project
This is my first real web development project. I built it to practice writing HTML, CSS, and JavaScript by hand, no templates, no shortcuts. Everything you see on the site was coded by me while learning.
I'm a CS student at IU International University, and this portfolio website is part of my commitment to grow consistently as a developer, one project at a time.

🗂️ File Structure
The project is organized into 3 separate files:

index.html — All the page structure and content
style.css — All the styling, animations, and layout
script.js — All the interactivity and JavaScript features


✨ Features I Built

Custom animated cursor with a trailing ring effect
Scroll reveal animations on every section
Animated skill bars that fill up as you scroll
SVG circular progress rings
Typing animation that cycles through different text
Project cards with a mouse-follow glow effect
3D tilt effect on stat cards
Floating badges in the hero section
Fully responsive layout for mobile and desktop
Navigation bar that shrinks and darkens on scroll
Staggered certification card fade-in animations
